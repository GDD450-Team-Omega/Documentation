Changes the texture of one of the four child tiles (one in each corner) so that it connects to adjecent path tiles. Since each child tile is in a corner, it is connected to two other tiles, the tile that comes first clockwise is the first connection, and the one 90 degrees clockwise is the second. Example: The top left corner is connected to the North and West tiles. West is considered the first connection, and North is the second.

Parameters
MatsA list of textures for the path tile.
FirstConnectionThe tile type in the first direction the corner connects to (clockwise).
SecondConnectionThe tile type in the second direction the corner connects to (clockwise).
OffsetThe position of the child tile in the ChildTiles array.
